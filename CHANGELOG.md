# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## v0.12.0 - 2026-01-06
### Added
* Add bot players to sessions - managers can add bots with selectable race and difficulty level
### Changed
* Refactored frontend: split update code into multiple files

## v0.11.0 - 2026-01-05
### Added
* Joinable sessions filter showing public/invited sessions with available player slots
* Default Neper server auto-created on first startup; warning shown when removing it; button to re-add it in Add Server dialog
### Changed
* Registration now auto-connects to server immediately; pending users can create races while awaiting approval

## v0.10.0 - 2026-01-04
### Changed
* Updated for neper API changes: session state enum (pending/started/archived), registration returns API key immediately, registration approval notifications, archive session functionality, and archived sessions filter with fetch button
* Optimized session notification handling to fetch only the updated session instead of all sessions
### Fixed
* Archived sessions now correctly display 'Archived' status instead of 'Not Started'

## v0.9.2 - 2026-01-03
### Added
* Added macOS builds to CI (Intel on macos-13, Apple Silicon on macos-14)
### Changed
* Updated macOS CI runners to macos-15-intel and macos-latest; set deployment target to macOS 13.0 for compatibility

## v0.9.1 - 2026-01-03
### Changed
* Added elm-review linting with NoUnused rules; removed dead code (StarsBrowser module, unused types/exports)

## v0.9.0 - 2026-01-03
### Added
* Added tooling for CL
* Remember and restore window position and size between sessions
* Generate animated GIF maps from session history in Map Viewer dialog
* Toast notification when API key is copied to clipboard
* Added stars in the browser, as an experimental feature.
### Changed
* Collapse Players section by default when session is started in session detail view
* Increase default window size to 1440x900 for better initial user experience
* Update houston dependency to v0.1.0
* Replaced 'View API Key' with 'Copy API Key' in user menu - now copies directly to clipboard without displaying
### Fixed
* Fix download icons not displaying properly on Backup and Historic buttons
* Fixed dialog accidentally closing when selecting text and releasing mouse outside dialog
* User menu now closes when clicking on the header/title bar
* GIF animation frame delay option now works correctly
* Fixed animated GIF map generation: planet names and options now correctly apply to all frames (uses houston v0.2.0 SetBaseData API)
